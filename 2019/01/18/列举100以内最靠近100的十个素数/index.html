列举100以内最靠近100的十个素数

标签（空格分隔）： 自编程序

n=100
c=0
while n>0:
flag=1
for i in range(2,n):
if n%i==0:
flag=0
break
if flag==1:
print(n,end=” “)
c+=1
n-=1
if c==10:
break
