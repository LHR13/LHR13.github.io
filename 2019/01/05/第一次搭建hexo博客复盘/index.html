一、安装node.js

  1. 去官网安装node.js，下好安装包正常操作即可。

  2. 打开cmd命令行，输入node -v和npm -v，出现下图证明成功。如果npm -v报错，很可能是没有一以管理员身份运行，电脑输入win+x选择管理员身份再次输入代码即可。

    二、安装git

  1. 去官网安装git ，下好安装包正常操作。
    1. 右键点桌面出现 Git GUL Here 和Git Bash Here 两行命令及证明安装成功。

      三、安装hexo

    2. 在认为合适的地方建一个文件夹，命名hexo（任意名字均可）
    3. 打开cmd命令行，输入d:，再输入cd hexo进入你所创立的那个文件夹。
    4. 输入npm install hexo -g，安装hexo npm install hexo -g，安装hexo。（注意，过程一定要联网）
    5. 输入hexo -v查看安装是否成功。如果出现下图则表示成功。
  1. 如果出现报错。如下图，则是运行hexo所需要的环境变量没有配置好，配置环境变量即可。方法：我的电脑右击——属性——高级系统设置——环境变量——选择选择系统变量里边的path——新建变量——复制粘贴以下内容 C:\Users\用户\node_modules\hexo\bin 即可。（注意，用户是你自己的用户名那个文件夹）

    三、正式生成博客

  2. 输入 hexo init
  3. 输入 npm install
  4. 输入 hexo g
  5. 输入 hexo s，hexo会为你自动生成一个网址如图。
  6. Github账户注册和新建项目，项目必须要遵守格式：账户名.github.io，不然接下来会有很多麻烦。并且需要勾选Initialize this repository with a README
  7. 进入安装hexo的那个文件夹里面，点右键选择Git Bash Here，进入到如图页面
  1. 输入 git config –global user.name “这里写你想给自己博客首页起的名字”。（注意空格）
  2. 输入 git config –global user.email “在这里输入邮箱”
  3. 输入cd ~/.ssh
  4. 输入ssh-keygen -t rsa -C “步骤十二你输入的邮箱”。完成后连续三个回车，生成密钥。
  5. 输入 eval”$(ssh-agent -s)”
  6. 输入ssh-add ~/.ssh/id_rsa
  7. 登陆Github，点击头像下的settings，找到 SSH and GPG keys 项目，点开，选择右上角的 New SSH key
  8. 在 C:\users\Administrator.ssh 文件夹里找到 id_rsa.pub 的文件，用记事本的打开方式打开它 ，将内容复制起来。
  9. 在网页内输入你的项目名 blog，在底下的大方框内粘贴上一步复制的内容。
  10. 在第6步打开的界面内输入 ssh -T git@github.com，在底下的代码中发现 HI+你的用户名就证明成功了。
  11. 在github网站上打开第五步你新建的项目，看到如图所示内容，点击方框右边将方框网址复制到剪贴板。（注意：右上角是 Use SSH）

  12. 在安装hexo的文件夹中，找到_config.yml 文件，用记事本方式打开在末尾将文件修改成以下格式
    （注意：rep 那一行就是你复制的内容）

  13. 在cmd中输入 npm install hexo-deployer-git –save

  14. 输入hexo new post “你好，hexo”
  15. 这时在文件夹 _posts 下已经能看到你创建的文件了
  16. 使用命令 hexo d -g 将你的文章生成以及部署在你的网站上
  17. 访问网站 http://用户名.github.io 就能看到你部署的文章了！
